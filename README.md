# 📝 React Todo App

A simple and responsive Todo App built using React to manage daily tasks.

## 🚀 Features
- Add todos
- Display todo list
- Delete & complete todosm
- All ,Active and Completed Buttons

## 🛠️ Tech Stack
- React (Vite)
- Custom CSS
- JavaScript
- GitHub for version control

## 🧑‍💻 Getting Started
```bash
git clone https://github.com/abuzarraziqgithub/todo-app.git
cd todo-app
npm install
npm run dev
```
